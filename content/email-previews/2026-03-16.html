#4 · IMPORTANTE · GitHub
Lightpanda: navegador headless escrito en Zig para agentes IA
Lightpanda es un navegador open-source construido desde cero en Zig para automatización y agentes IA, ofreciendo 9x menos memoria y 11x más velocidad que Chrome. Compatible con Playwright y Puppeteer, representa una alternativa eficiente para scraping y testing a escala.

#5 · IMPORTANTE · GitHub
LangChain lanza Deep Agents: harness preconstruido para agentes
Deep Agents es un framework listo para usar de LangChain que elimina la necesidad de cablear prompts y herramientas manualmente, incluyendo planificación, acceso a filesystem y shell. Disponible en Python, acelera el desarrollo de agentes autónomos con sensatos defaults.
